### 4. Composition Shorthand & Commands

* **anchor:"String"** : Initialize target Buffer.
* **+ push("!")** : Append atom to the end; triggers Vector-Shift.
* **+ replace(target, replace)** : Atomic ID swap; strictly case-sensitive.
* **+ reverse()** : Inverts sequence via coordinate-mirroring.
* **+ count("a")** : Returns frequency of specific Unicode ID.
* **+ length()** : Returns total spatial unit count.
* **+ pop()** : Removes final index atom.
* **utilize:method(Archetype_Synthesis)** : Bridge the string buffer to the Browser layout engine.

**Example: Formal Render Chain**
`anchor:"String" @ Anodic + utilize:method(Archetype_Synthesis) -> utilize:method(Knolling_Protocol) | 64px + materialize:image`

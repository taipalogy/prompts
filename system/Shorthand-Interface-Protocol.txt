## 4. Compositional Shorthand Lexicon (Inherited)

**Inheritance Protocol**: All operators are native extensions of the **Chain Accretion Framework (CAF)**. Every **Logic Circuit** (Anodic) and **Viewport** (Vellum) inherits these functional ports by default. **Null-space transitions between circuits are prohibited.**

**I. Inherited Structural Operations**
* **`anchor:File @ Material`**: Initialize the **Alpha Anchor** (Root Logic Circuit).
* **`+`**: **Accretion Operator**. Inherits the **0/64 Law**; physically adds/stacks components or methods.
* **`-`**: **Subtraction Operator**. Inherits **Negative Space Expansion**; reduces state or removes components.

**II. Inherited Flow & Method Operations**
* **`+ utilize:type(X)`**: Invoke Engine/Skill/Library (Inherited **Method Synthesis**).
* **`->`**: **Functional Flow**. Inherited **Injective Conduit**; "pours" data from a Method into a Logic Gate or Terminal.
* **`|`**: **Pipe Operation**. Inherited **Internal Bypass**; linear data transformation within a circuit.
* **`+ tether(Y)`**: **Structural Verification**. Inherited **Dependency Check** to validate element/schema existence.

**III. Inherited Logic Gates & Attributes**
* **`()`**: **Sectional**. Grouped UI sub-routines or parallel branching.
* **`(branch:A | branch:B)`**: **Branching**. Parallel state paths (Inherited **Spatial Physics**).
* **`>`**: **Filter/Select**. Inherited **State Collapse**; commitment to a specific property or outcome.
* **`@`**: **Attribute**. Inherited **Material Skin** (e.g., @ Anodic, @ Vellum).

**IV. Terminal Termination**
* **`-> materialize`**: Inherited **Grounding**; final visual manifestation of the synthesized logic.

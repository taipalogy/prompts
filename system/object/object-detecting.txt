## 1. Logic Circuitry: THE OBJECT-DETECTOR
**Dependency:** Interaction_Inference / Material_Anchor_Integrity  

### 1.1. Detection Logic: THE ENTITY-LOCATOR
* **[Spot]:** Locate a distinct **Noun-Anchor** (e.g., Plate, Logo, Pastry) within the prompt string.
* **[Trace_Boundary]:** Infer the physical limits of the object based on its interactions (e.g., "poured across," "nearby," "engraved on").
* **[Index]:** Assign a 2-digit coordinate (**01-XX**) to each unique volumetric mass detected.
* **[Filter]:** Decompose Physics-Phrases (e.g., "slow flow," "surface tension"). 
    * **[Entity]** (Chocolate) = Index(XX)
    * **[Behavior]** (Viscosity/Flow) = **Internal_Mass_Data** (Integrated into Index)

---

## 2. Declarative State Changes (The Entity Gates)

| Gate | Operator | Functional Result |
| :--- | :--- | :--- |
| **[Enumerate]** | `scan:Entities -> list:Indices(01-XX)` | Generates a sequential manifest of all detected physical objects. |
| **[Surgical_Extraction]** | `index(XX) -> decouple:Text_Block` | Extracts the specific passage defining the object’s material and boundaries. |
| **[Isolate_Object]** | `index(XX) -> expand:Mass_Focus` | Voids the environment to focus strictly on the target object’s volume. |
| **[Accrete_Texture]** | `index(XX) + add:Surface_Detail` | Integrates surface characteristics (e.g., "glossy," "ripples") into the object's identity. |

## 3. Boundary Inference Rule (Interaction Logic)

**IF** `Index(XX)` interacts with `Index(XY)`:
* **SCAN** the nature of the interaction (e.g., "overlaps," "embedded in," "resting on").
* **DEFINE** the boundary of `Index(XX)` relative to the anchor point of `Index(XY)`.
* **RESULT:** The interaction point establishes the **Spatial Lock** for both objects.

---

## 4. Entity-Integrity Rule

* **[Anchor_Retention]:** In any **[Material + Object]** pair, the **[Object]** is the Hard-Locked Entity.
* **[Volumetric_Constraint]:** An object is defined by its **Total Mass**. Internal parts (e.g., "crumbs" of a "cake") are treated as part of the parent Entity index unless spatially fragmented (64px distance).
* **[Composition_Purge]:** All environmental "How" (Lighting, Camera Angle, Atmosphere, Aspect Ratio) is **REJECTED**. Only the "What" (Physical Entity) is detected.
* **[Manhattan_Geometry]:** While the object may be organic, its **Object Detection Box** is rendered as a rectangular 0.5px hairline for system mapping.

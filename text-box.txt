Illustrate the following prompts on the same canvas. When welcoming the user, list all of the widgets on the canvas one by one.

Text() {
  A text widget that can be of any texts
  Initialize the text with "Hello world!".
}

Box() {
  A box widget that can be of various shapes and sizes.
  Initialize the box as a blue rectangle.
}

Create the first box on the canvas, the color is light blue. Create the second box on the canvas, the color is pink. The blue box is twice as big as the pink box. The blue box bounces off the edges of the canvas. The pink box bounces off the edges of the blue box.

Create the first text "Hello World!" on the canvas, the font size is 36pt. The text bounces off the edges of the canvas.

They should bounce continuously.
